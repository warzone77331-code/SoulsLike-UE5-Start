# SoulsLike UE5 Starter

A Souls-like prototype built in Unreal Engine 5.

## Features
- Bonfire checkpoint system
- Enemy AI (basic aggro + attack)
- Boss fight system
- Shortcut gates
- Stamina + combat system

## Setup

1. Clone repo
2. Open SoulsLike.uproject in Unreal Engine 5
3. Open map: Content/SoulsLike/Maps/L_RuinedCastle

## Controls

- LMB: Attack
- Space: Dodge
- E: Interact
